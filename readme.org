** Pour commencer
*** Pré-requis :

**** Java : =java=

***** Linux:

      [[https://www.java.com/fr/download/help/linux_install.html][Installer Java sur Linux]]

*** Installation

    Assurer vous d'avoir la dernière version compilée :

    =make=
